This project is a web application for a microblogging platform that was created for the CSCI4370 course. With features like hashtags, comments, likes, bookmarks, and a follow system, users may produce posts, communicate with other users, and interact with content.

Using database concepts like ER modeling, relational schema design, SQL queries, and database-driven application development in a real-world application was the aim of this project.
